HTML

  • The head element contains information about the webpage.
  • The body element respresents the visible content shown to the user.
  • The img element contains information relative to an image.
  • The main component holds the structure of a webpage.
  • A class is used to specify an element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can use class selectors to apply the same properties to any HTML assigned the class.
  • We can use height and width descriptors to change such on elements of our webpage.
  • Text align allows us to align or text as desired.

Git

  • The cd command allows you to navigate through the directory.
  • The cd .. comand navigates us back one step in the directory.
  • The ls command displays the contents of the directory.
  • The pwd command prints the working directory.
  • Any git specified commands start with git.
  • git pull origin main will populate our local branch with the remote branch.
  • git checkout -b BRANCHNAME adds a new branch to our local environment.
  • git add -A stages our changes.
  • git commit -m "Message here" commits are changes with a descriptive message.
  • git push origin BranchName pushes our current branch to our remote branch.
  • The command touch fileName creates a new file in the given directory.

Javascript

  • To declare a variable we use the syntax var variableName.
  • To create a function we use the syntax function functionName().
  • console.log() will display what is within the parenthesis.
  • To declare an array we do so as such var variableName = ["v1", "v2", "v2"
  • Arrays are not limited to strings.
